# UI-UX-NTU

Abot Kamay WiFi 官方網站 — 菲律賓低月費 WiFi 服務平台。

## 簡介

Abot Kamay Digital Inc. 提供廣告驅動的低月費 WiFi 服務，連結場館、用戶與廣告主三方：

- **場館 (Venues)** 提供 WiFi 服務
- **用戶** 觀看短廣告即可上網
- **廣告主 (Advertisers)** 觸及精準受眾

## 專案結構

```
├── index.html              # 首頁
├── about.html              # 關於我們
├── venues.html             # 場館頁面
├── advertisers.html        # 廣告主頁面
├── advertiser-inquiry.html # 廣告主諮詢表單
├── contact.html            # 聯絡我們
├── privacy*.html           # 隱私權政策 (EN/TL/ZH)
├── terms-of-service*.html  # 服務條款 (EN/TL/ZH)
├── css/                    # 樣式表
├── js/                     # JavaScript (動畫、i18n、主程式)
├── portal-preview/         # Captive Portal 預覽
├── legal/                  # 法律文件
└── Abot Kamay Brand Kit/   # 品牌素材 (Logo SVG)
```

## 技術

- 純 HTML / CSS / JavaScript（無框架）
- 多語系支援 (English、Tagalog、繁體中文)
- RWD 響應式設計
